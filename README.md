## Local Setup

### 1) Clone Repository
```bash
git clone <YOUR_REPOSITORY_URL>
```
### 2) Go to project directory
```bash
cd mini_vehicle_management_system_frontend
cd logistix360_VMS_FE
```

### 4) Get the latest update from main branch
```bash
git pull
```

### 3) Checkout development
```bash
git checkout development
```

### 4) Get the latest update
```bash
git pull
```

### 4) Install the dependencies
```bash
npm install
```

### 5) Run the project
```bash
ng serve
```
